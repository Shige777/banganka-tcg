## 1. 世界の核

- 舞台は `交界`——次元と時間の結節点。宇宙規模の「交差路」
- 願いを叶える奇跡は `万願果`
- 交界には無数の世界・時代・星系から存在が流れ込む
- 人々はそれぞれ異なる願いを抱えて現れる
- `交界` を作り出した上位存在として `界主` が存在する

### 交界の物理的性質（SF設定）

交界は「どこでもない場所」ではなく、**次元間の構造物**として存在する。

- 異なる時代・世界・星系をつなぐ「次元の折り目」が交差する地点
- 自然発生ではなく、**界主によって設計・構築された人工構造物**
- 交界の内部では時間が通常と異なる流れ方をする（外の世界では時間が止まっている）
- 交界に流れ込む存在は、自分の世界では「消えた」扱いになる

### 各世界のSF要素

交界に繋がる世界はそれぞれ**固有の文明・技術・宇宙観**を持つ。
「魔法」と「科学」の境界は各世界によって異なる。統一したエネルギー系は存在しない。

| 願主 | 世界のSF要素 |
|------|------------|
| **灯凪**（古代・都市国家） | 星系移民の第一世代が建てた入植都市。精密な天文観測技術と口承記録システムを持つ。本星との通信が途絶した孤立植民地。滅びは「外敵」ではなく、宇宙的な孤立と内部崩壊だった |
| **Aldric**（中世・征服王朝） | 次元の「亀裂」を武器化した征服技術を持つ世界。Aldricの軍は異なる空間から奇襲できた。ただしその技術は制御不能で、使うたびに世界を少し壊していた |
| **崔鋒**（近世・革命期） | 「記憶の結晶化」技術が発達した世界。人の記憶を固体に刻み保存できる。革命軍はその技術で過去の「歴史」を書き換えた。崔鋒はその技術の被害者 |
| **Rahim**（近代・産業崩壊期） | 次元共鳴を利用した重工業が普及した世界。効率的だが次元干渉による構造崩壊を起こしやすい。姉が死んだ工場事故は、次元共鳴の制御ミスが原因だった |
| **Amara**（近未来・文明崩壊後） | 複数の星系に広がった人類文明が、AI設計の都市管理システムに移行した世界。Amaraはその中核となる都市AIのアーキテクト。最適化され続けるシステムは、人の「逸脱する余地」を消した |
| **Vael**（不明・交界の住人） | 出自なし。Vaelは交界そのものから生まれたか、あるいは界主が設計した最初の「テスト存在」の残滓かもしれない。どの世界の記録にも存在しない |

## 5. 界主

### SF設定

界主は「神」ではなく、**宇宙規模の知性体**として設計する。

- 無数の次元・時代・星系を観測し、それらを繋ぐ `交界` を設計・構築した存在
- かつては単一の完全な意志を持っていたが、**ある事象によって意識が分裂した**
- 分裂した欠片たちは互いに矛盾し、果求者を異なる目的で動かそうとしている
- 界主自身も「叶えられなかった願い」を持っていたとされる
- `ナル` は界主の断片のひとつが具現化した使い、という可能性がある

### 分裂の意味（SF的解釈）

| 解釈 | 内容 |
|------|------|
| AI的 | 界主は宇宙文明が作った超高度知性。ある判断エラーにより自己矛盾が生まれ、意識が分裂した |
| 物理的 | 界主が扱った力が限界を超え、意識が多次元に飛散した |
| 意志の崩壊 | 「叶えられなかった願い」が原因で内部から崩壊した |

### 物語上の機能

- 界主の過去が交界の構造・万願果の存在理由と直結する
- 終盤、果求者が界主の「欠片」を目の当たりにする
- 「願いを持ち続けた知性」と「願いに共鳴する存在」が対峙するクライマックス
- ナルが界主の使いであることが判明したとき、ナル自身の「どちら側にいるか」が問われる

## 6. 果求者（かぐしゃ）

> 詳細は `PLAYER_CHARACTER.md` を参照。以下はSTORY_BIBLE内の要約。

### 基本設定

| 項目 | 内容 |
|------|------|
| 真名 | プレイヤーが入力する |
| 呼称 | 果求者（交界における役割名） |
| 出自 | 不明。交界に気づいたら立っていた |
| 記憶 | なし |
| 外見 | 各時代の空気に馴染む「時代の鏡」。目だけは変わらない（静水の眼） |

### 核心：共鳴の感覚

果求者には確定した過去を持たせない。
代わりに、**真の願いに共鳴する感覚**だけを持つ。

願いを聞いた瞬間、内側で何かが鳴る。感情ではなく「認識」。
考えるより先に、体が向いている。果求者はこの感覚を「慕い」と呼ぶ。

### 物語上の機能

- 各章で願主と向き合い、その願いを共鳴によって「本物」と確認する
- プレイヤーの投影先として、感情の余白を最大に保つ
- 終章（Vael）で初めて「自分の慕いの向かう先」を問われる
- 答えは出さない——次の問いへと委ねられる

### 果求者の語録（トーン参考）

> 「なぜここに来たのか、俺にはわからない。ただ、お前の願いが鳴っていた」

> 「叶えてやれるかどうかはわからない。でも、本物だということは知っている」

### 7.1 灯凪（とうな）

| 項目 | 内容 |
|------|------|
| 文化圏 | 和風 |
| 出自時代 | 古代・都市国家 |
| 願相 | 穏（#59C36A） |
| 情相 | 慈愛（Grace） |
| 性別・年齢 | 少女（14〜16歳相当） |
| 果求者との関係 | **聞く者** |

**願い**：「私が語った物語を、誰かひとりに届けたい」

**背景**：
盲目の語り部。戦火に滅んだ都市国家の生き残り。
死者の声を聞き、言葉にして生きてきた。
しかし都市が滅んだとき、語りを聞いた人間は全員死んだ。

英雄的な復讐でも歴史の改変でもない——**ただひとりに届けばいい。**

#### キャラクターギャップ（表と裏）

灯凪の本質は**「最も優しい能力の裏に、最も容赦ない目を持つ盲目の少女」**。
ゲーム上は味方全体を守り癒す最も「優しい」リーダーだが、性格は全キャラで一番容赦がない。

| 表（ゲーム・戦場） | 裏（物語・素顔） |
|---|---|
| 味方全体をバフで守る | 口が悪い。敬語を使わない。容赦なく本質を突く |
| HP回復＝癒し手 | 「あんた、嘘ついてるでしょ」——癒すのではなく暴く |
| Blocker＝盾になる | 譲れないことにはテコでも動かない。全キャラ一番頑固 |
| 最も初心者向けの安定型 | 怒ると声が低くなり、盲目なのに「見る」。全キャラで一番怖い |

**設計意図**：「優しさ」と「容赦のなさ」は矛盾しない。灯凪は嘘をつかないし、嘘を許さない。それが一番残酷で、一番優しい。目が見えないから表情で騙せない。声だけが真実。

#### 性格

- タメ口。敬語を使わない。年上にも「あんた」
- Aldricに「おっさん、暗い顔してんなよ。声でわかるんだよ」
- Vaelに「あんた、笑ってるふりしてるでしょ。声が嘘ついてる」
- 同情されるのが大嫌い。「かわいそう？ は？ どこが」
- 甘えるのが下手。助けてもらっても「べつに」。でも耳が赤くなる
- 怒ると声が低くなり、閉じた目が相手の方を向く。「見えないはずなのに見ている」

#### 外見デザイン

**体格**：小柄で華奢。だが姿勢がいい。語り部として人前に立ってきた者の立ち姿。

**顔・髪**：
- 閉じたままの目。まぶたの上に薄い傷跡（生まれつき盲目）
- 長い黒髪を和風に結っているが崩れかけている（自分では結えない）
- 穏（＝緑）とのリンクは帯や衣の差し色で

**服装（ギャップ）**：
- バトル時：白い巫女装束。語り部の「公の姿」
- 素：着物がずれたただの少女。誰かが直すと「べつに」と言いつつ大人しくする

**象徴モチーフ**：
- 片耳の金の耳飾り（語り部の証。もう片方は滅んだ街に残してきた）
- 杖を持たない（音と気配だけで歩く。異常な知覚力の証）
- 裸足（足裏で世界を感じている）

#### ボイスイメージ

- 少女の声だが甘さが少ない。滑舌がいい（語り部だから）
- 語りモードになると声が完全に変わる。低く、遠く、何百年も前の声を再生するような響き
- 怒ると一段低くなる。静かだが逃げ場がない
- CV方向性：早見沙織系

#### 果求者との関係

灯凪は盲目なのに、果求者の目を「見た」と言った。
「あんたの目、ずっと何かを探してるみたいだ」——盲目の少女だけが、静水の眼に気づく。
果求者が灯凪の語りを最後まで聞く。それだけで願いは叶う——万願果を使わずに。
灯凪は泣いているとき自分では気づかない。目が見えないから。果求者が頬に触れて初めて知る。

#### ナルとの関係

ナルは灯凪の語りを黙って聞き続け、終わってから「なんか泣けてくるな」とこっそり言う。
灯凪はその声を聞いている。聞いているが、何も言わない。

**象徴的なセリフ**：
> 「あんた、なんで笑いながら戦ってんの。怖いんでしょ、ちゃんと」

### 7.2 Aldric（アルドリック）

| 項目 | 内容 |
|------|------|
| 文化圏 | 欧風（古英語「高貴な支配者」） |
| 出自時代 | 中世・征服王朝 |
| 願相 | 曙（#FF5A36） |
| 情相 | 悔恨（Remorse） |
| 性別・年齢 | 壮年男（35〜40歳相当） |
| 果求者との関係 | **鏡（裁かない者）** |

**願い**：「俺が踏み潰した命の数だけ、俺には戦う義務がある」→ 滅ぼした民を救いたい

**背景**：
大陸を征服した王。強さのために民を犠牲にした男。
万願果で滅ぼした民を蘇らせたい——だがそれは本当に民のためか、自分の罪悪感を消すためか。
自分でもまだ答えが出ていない。

#### キャラクターギャップ（表と裏）

Aldricの本質は**「戦場の鬼神 × 素は静かで不器用な男」**である。
ゲーム上は最も暴力的な攻撃型リーダーだが、物語上の素顔はそれと真逆。

| 表（ゲーム・戦場） | 裏（物語・素顔） |
|---|---|
| 全軍突撃を命じる暴君 | 声が低くて静か。怒鳴らない。命令も穏やかな口調 |
| 「覇王の一喝」（Lv3スキル） | 本人は叫ぶのが苦手。このスキル発動が唯一声を張る瞬間 |
| 100の民族を滅ぼした征服王 | 征服した街で最初にしたのは孤児院と庭園を作ること |
| 冷徹な戦場判断 | 感謝されると目を逸らす。褒められると照れる |
| 誰も逆らえない威圧感 | 花を育てるのが好き。人と距離を詰めるのが下手 |
| 感情を見せない鉄面 | 果求者に「お前の願いは本物だ」と言われて初めて目が赤くなる |

**設計意図**：情相は「悔恨」だが、行動は「慈愛」に見える。このズレがAldricの深みであり、プレイヤーに「この男の本質は悔恨なのか慈愛なのか」を考えさせる。

#### 性格

- 普段は寡黙。声が低く、話すとき間がある（「……行くぞ」）
- 感情を表に出さないが、押し殺しているのではなく表現が不器用なだけ
- 罪の重さを感情ではなく事実として正確に理解している。だから言い訳をしない
- 人懐っこさは自覚していない。「裁いてくれ」と言うのも、本当は誰かにそばにいてほしいから
- 孤児院を作っていたことは誰にも言わない。聞かれても「……別に」としか答えない

#### 外見デザイン

**体格**：大柄だが筋肉質というより「骨太で厚い」。威圧感は体格ではなく佇まいから来る。

**顔つき**：
- 征服王のイメージと裏腹に、穏やかで整った顔立ち
- 目の下に隈がある（夜ごと夢に踏み潰した命たちを見るため眠れない）
- 無精髭。王なのに身だしなみに無頓着

**髪**：暗い赤髪を無造作に後ろで束ねている。戦場では解ける。曙（＝夜明けの赤）との色リンク。

**服装（ギャップ）**：
- 戦場：重厚な黒鉄の鎧＋深紅のマント。覇王の威圧
- 素：鎧を脱ぐと質素なリネンのシャツ。袖をまくって庭の手入れをしている。甲冑の中身がこれかと驚くほど柔らかい印象

**象徴モチーフ**：
- 片手に花の種を握っている（征服した土地に庭園を作る習慣の象徴）
- 鎧の裏側に子供の落書きがある（孤児院の子がこっそり描いた。消せなかった）

#### イラスト方向性

| 用途 | 構図 |
|---|---|
| リーダーカード（メイン） | 黒鉄鎧＋深紅マント。右手に大剣、左手は拳を握っている——よく見ると拳の中に花の種を握っている |
| Lv2スキル演出（突撃令） | 静かに片手を上げるだけ。叫ばない。それだけで全軍が動く |
| Lv3スキル演出（覇王の一喝） | 唯一声を出す瞬間。目を見開いて一言。普段静かな男の叫びだからこそ震える |
| ストーリー立ち絵 | 鎧なし。リネンのシャツ姿で庭に座り、目を伏せている。傍にナル |
| 敗北演出 | 膝をつくが、花の種を地面に押し込むように手を置く |

#### ボイスイメージ

- 低音で落ち着いたトーン。早口にならない
- 命令口調ではなく「……行くぞ」のように間がある話し方
- 感情が溢れる瞬間だけ声が震える（Lv3スキル発動時、HP閾値を割った時）
- CV方向性：中村悠一系（知的で柔らかいが芯が強い）

#### 果求者との関係

果求者は裁かない。どんな罪を持つ者の願いも「本物だ」と告げる——Aldricにとってそれが一番苦しい。
「裁いてくれるなら、まだ楽だ。お前が黙って立っているのが怖い」
果求者に「お前の願いは本物だ」と言われたとき、初めて目が赤くなった。

#### ナルとの関係

ナルはAldricを最初「でっかい、こわ」と評する。
話してみると「意外と真面目じゃん」と口数が少なくなる。
Aldricが庭の手入れをしているのを見て「……王様なのに？」と聞くと、Aldricは「……別に」とだけ答える。ナルはそれ以上聞かない。

**象徴的なセリフ**：
> 「悔恨か自己満足か——そんなことはどうでもいい。俺の手が届く限り、俺は戦う」

### 7.3 崔鋒（ツェイ・フォン）

| 項目 | 内容 |
|------|------|
| 文化圏 | 中華風（「崔」＝高い山、「鋒」＝刃先） |
| 出自時代 | 近世・革命期 |
| 願相 | 玄（#3A3A46） |
| 情相 | 矜持（Dignity） |
| 性別・年齢 | 老将（60代後半相当） |
| 果求者との関係 | **記録者** |

**願い**：「勝者が歴史を書く。だが俺は、敗者の真実を残す」

**背景**：
革命に敗れた老将。歴史から抹消されようとしている側の人間。
万願果で正しい歴史を後世に刻みたい。
力ではなく、記録を残すための戦い。

#### キャラクターギャップ（表と裏）

崔鋒の本質は**「最も守備的な能力者 × 最も口が悪いファンキーなクソジジイ」**。
ゲーム上は壁を立てて守る最も受動的なリーダーだが、素は全キャラで最も攻撃的な口を持つ老人。

| 表（ゲーム・戦場） | 裏（物語・素顔） |
|---|---|
| 壁を立てて守る。自分からは攻めない | 口が悪い。遠慮しない。年寄りの特権で好き放題言う |
| 殉陣＝黙って部下を送り出す | 「死にたきゃ死ね。だが意味のある死に方をしろ」 |
| ゲーム中最も受動的なリーダー | 全キャラで最も攻撃的な口。フィルターなし |
| Blocker壁＝静かな守護者 | 「守ってやってるんじゃねえ。お前らが勝手に壁の後ろに来るんだ」 |

**設計意図**：ゲームでは一番おとなしい（壁を立てて待つ）のに、素は一番口が悪い。口は最悪だが言っていることは全部正しい。長い人生で嘘も虚勢も見すぎた。容赦がないのは、嘘のまま死んでいった部下を知っているから。

#### 性格

- 全キャラで最も口が悪い。年寄りの特権。遠慮しない。全部正論
- 嘘が嫌い。言い訳が嫌い。キレイゴトが一番嫌い。「キレイに言うな。汚く言え。本当のことは汚い」
- 料理が上手い（野営飯）。若い願主に食わせる。「腹が減ってるから機嫌が悪いんだ。食え」
- Rahimにだけ口調が柔らかい。本人は気づいていない。周りは気づいている
- 灯凪とは唯一の口喧嘩相手。「クソジジイ」「口の減らないガキ」。互いに楽しんでいる

#### 外見デザイン

**体格**：大柄で骨太（180cm）。筋肉は落ちているが骨格が立派。老いても将軍の体。

**顔・髪**：
- 深い皺。鋭い目。目だけが燃えている
- 白髪混じりの黒髪を無造作に後ろで束ねている
- 顎ひげ。無造作に見えて最後の一線は揃えている（矜持の視覚化）

**服装（ギャップ）**：
- バトル時：古びた旧軍の鎧。大盾。紋章は革命軍に削られた跡
- 素：鎧を脱いでただの老人。腕まくりで料理。エプロンなし

**象徴モチーフ**：古びた将軍の鎧（これを着てた奴らが消えた）、顎ひげ（矜持）、傷だらけの手（料理も戦いも同じ手）

#### CV方向性

大塚明夫系（低音で重厚。飄々としたふざけ方）or 中田譲治系（威厳と悪戯っぽさの同居）

**果求者との関係**：
崔鋒は果求者に「記録者」として接する——誰かが聞いていてくれることが、この老将にとって唯一の勝利だから。
果求者が黙って聞き続けることで、崔鋒の語りに「残る力」が与えられる。
ナルを一度だけ真剣な目で見た。ナルは「俺、見透かされた気がした」と固まった。崔鋒は「面白い獣だ」とだけ言った。

**象徴的なセリフ**：
> 「お前が残さなければ、誰も知らない。それだけでいい——ただ、聞いていてくれ」

### 7.4 Rahim（ラヒム）

| 項目 | 内容 |
|------|------|
| 文化圏 | アラビア風（「慈悲深い者」の意） |
| 出自時代 | 近代・産業崩壊期 |

| 情相 | 追慕（Longing） |
| 性別・年齢 | 少年（14〜15歳相当） |
| 果求者との関係 | **問われる者** |

**願い**：「死んだ姉を蘇らせたい」

**背景**：
産業崩壊期の少年。姉を事故で失い、万願果で蘇らせたいと交界に来た。
純粋な動機——ただ、死者を蘇らせることの意味は、言葉にならない重さをはらんでいる。

「慈悲深い者」という名が皮肉でもあり、願いの純粋さの証でもある。

#### キャラクターギャップ（表と裏）

Rahimの本質は**「何度でも蘇らせるトリックスター × 一番大切な人だけは蘇らせられない、真っ直ぐな少年」**。
ゲーム上は顕現をバウンス→再展開する最も「ずる賢い」リーダーだが、素は全キャラで一番素直で嘘がつけない少年。

| 表（ゲーム・戦場） | 裏（物語・素顔） |
|---|---|
| 何度でも蘇らせる「再演」の達人 | 姉だけは蘇らせられない。一番大切な人だけ戻らない |
| 常に手札が豊富で選択肢に困らない | 本当に欲しい選択肢（姉の生存）だけがない |
| トリッキーで器用。対戦相手を翻弄 | 嘘がつけない不器用な少年。真っ直ぐすぎて痛い |
| ゲーム中最も「ずる賢い」リーダー | 全キャラで一番素直。策略がない |

**設計意図**：ゲームで「蘇らせる」たびに、プレイヤーはRahimが本当に蘇らせたいものを思い出す。6人中最もメカニクスがキャラの切なさに直結。

#### 性格

- 姉の口癖「誰かを助けられる人間になれ」を文字通り守っている。考えるより先に体が動く
- 基本的に明るくよく笑う。Vaelとは親友。一緒に走り回る
- 夜になると黙る。寝つきが悪い。工具ベルトを胸に抱いて眠る
- 姉のことを聞かれると明るいまま答えるが、語尾が震える
- 食事に無頓着（Amaraに管理される）。髪は自分で不揃いに切る（姉が最後に切った長さを変えたくない）

#### 外見デザイン

**体格**：小柄で痩せている（155cm）。栄養不足の工場街の少年。でも動きは速い。

**顔・髪**：
- 浅黒い肌。アラビア系。目が大きく年齢より幼く見える
- 黒髪を自分で不揃いに切っている。姉が切ってくれていた名残
- 金色の瞳（遊＝#F4C542）。光が当たると姉と同じ色に見える

**服装（ギャップ）**：
- バトル時：工場の作業着ベースの軽装。袖まくり。腰に姉の工具ベルト
- 素：同じ服だが汚れ、膝に絆創膏、靴に穴。工具ベルトだけ綺麗に手入れ

**象徴モチーフ**：姉の工具ベルト（唯一の形見。常に身につけている）、不揃いの髪（姉が切った長さから変えたくない）、金色の瞳（姉と同じ色）

#### CV方向性

村瀬歩系（少年感と芯の強さ。声の奥に寂しさ）or 山下大輝系（真っ直ぐな少年声。熱くなると震える）

**果求者との関係**：
Rahimの願いを共鳴した瞬間、果求者は初めて「叶えていいのか」と自問する。
共鳴が「本物だ」と告げていても、それが正しいかどうかは別の問い——果求者の「慕い」の限界が初めて見える章。
ナルが「俺もそういう気持ち、あるかも」と初めて本音をこぼす。Rahimとは一緒にナルの尻尾を追いかけ回す仲（Vaelと二人で。ナルは「うるさい二人組」と呆れるが嫌いじゃない）。

**象徴的なセリフ**：
> 「姉ちゃんは俺に、誰かを助けられる人間になれって言ってた。だから俺は戦う」

### 7.5 Amara（アマラ）

| 項目 | 内容 |
|------|------|
| 文化圏 | アフリカ系（「永遠」「不滅」の意） |
| 出自時代 | 近未来・文明崩壊後 |
| 願相 | 妖（#9A5BFF） |
| 情相 | 諦観（Resign） |
| 性別・年齢 | 青年女（25〜28歳相当） |
| 果求者との関係 | **証人** |

**願い**：「私が設計したこの世界を、根こそぎ消したい」

**背景**：
文明崩壊後の近未来から来た建築家・設計者。
自分が作った都市システムが人々を苦しめ続けている。
万願果で自分の創造物を消したい——「永遠」という名を持つ女が、永遠を消そうとしている。

#### キャラクターギャップ（表と裏）

Amaraの本質は**「冷徹な破壊者 × 作ることをやめられない情熱的なクリエイター」**。
ゲーム上は自軍を冷徹に解体するコンボプレイヤーだが、素は自分の作品を語ると止まらない情熱的な設計者。

| 表（ゲーム・戦場） | 裏（物語・素顔） |
|---|---|
| 自軍を冷徹に解体する | 自分の作品を語ると止まらない情熱的なクリエイター |
| 「世界消去宣言」で全てを消す | 消す直前に自分の設計の美しさを思い出して躊躇する |
| 残骸を力に変える合理性 | 交界でも手が止まらず何かを作り続ける |
| ゲーム中最も冷たいコンボ | 他の願主の生活を「設計」して面倒を見てしまう |
| 計算された破壊 | 丁寧語。崩さない。壊す瞬間も「では、消しますね」 |

**設計意図**：「壊す者」と「作る者」は同一人物。Amaraは世界を消したいのではなく、「完璧に作れてしまう自分」が怖い。作ることをやめられない人間が、作ったものを消さなければならない——その苦しみをプレイヤーはゲーム体験（自壊コンボ）で追体験する。

#### 性格

- 全キャラ中最も丁寧語。「ですます」を崩さない。設計者としての矜持
- 落ち着いていて冷静に見えるが、何かを作っているときだけ表情が崩れる
- 人と話していても手元で何かを組み立てている。指が止まらない
- 他の願主の導線が気になって仕方ない。Vaelの服を直し、Rahimの食事を管理し、灯凪の動線を確認する
- 都市の設計を語り出すと止まらない。目が輝いて早口になる。その直後に「……もう存在しない都市の話をしても仕方ないですね」と黙る

#### 外見デザイン

**体格**：長身（174cm）でスタイルがいい。姿勢が完璧。自分の体すら「設計」しているかのような正確さ。

**顔・髪**：
- 褐色の肌。アフリカ系の美しさ。知的で整った顔立ち
- 白に近い銀髪をショートカットに切り揃えている。「効率的だから」
- 紫の瞳（妖＝#9A5BFF）

**服装（ギャップ）**：
- バトル時：黒い長衣に紫のラインが走る。設計ツール（細い杖）を手にしている
- 素：袖をまくっている。手にインクの跡。砂や石の粉がついている

**象徴モチーフ**：設計ツール（作る道具で壊す）、指先のインク跡（作ることをやめられない）、砂の建築モデル（作っては壊す）

#### イラスト方向性

- メイン：黒い長衣で空中に紫色の設計図を描く。背景に崩壊する都市のシルエット
- ストーリー立ち絵（通常）：袖をまくってインク跡の手。何かを設計中の微笑み
- ストーリー立ち絵（語り）：目を輝かせて早口。ジェスチャーが大きい。普段の冷静さが消えている
- 信頼度MAX：砂の小さな都市の横に座っている。壊していない。初めて壊さなかった朝

#### CV方向性

坂本真綾系（知的で柔らかいが芯がある。丁寧語と情熱のギャップ）or 沢城みゆき系（クールと熱のスイッチが鮮やか）

**果求者との関係**：
Amaraは果求者を「ただ見ている者」と言い切る。「あなたはただ見ているだけ。それが怖くないの」
果求者は答えない——怖いのかどうか、自分でもわからないから。分かり合えないまま、それでも果求者はそこに立ち続ける。
ナルはAmaraを苦手としつつ「なんか俺のこと分析してる気がする……」と評する。Amaraはナルを見て「かわいい構造ですね」と言う。

**象徴的なセリフ**：
> 「あなたが支えたいのは私の願いじゃない。私が戦う姿でしょう。それは支援じゃなく、消費よ」

### 7.6 Vael（ヴァエル）

| 項目 | 内容 |
|------|------|
| 文化圏 | 創作語（由来なし・意図的な空白） |
| 出自時代 | 不明・交界の住人 |
| 願相 | 空（#4DA3FF） |
| 情相 | 渇望（Thirst） |
| 性別・年齢 | 不詳（見た目20代前半） |
| 果求者との関係 | **鏡の向こう（問いの反射）** |

**願い**：「俺はいつからここにいる？　俺を呼んだのは、誰だ？」→ 誰かに「存在した」と認めてほしい

**背景**：
交界の住人。出自時代不明。記憶がない。
由来のない名前を持つ——存在証明を求める者が、語源のない名を持つことは意図的な空白。
万願果に求めるのは「自分が存在した」という証明。誰かの記憶に残ること。

#### キャラクターギャップ（表と裏）

Vaelの本質は**「存在を消す冷たい能力者 × 一番うるさくて一番寂しい奴」**。
ゲーム上は相手の場を封じ、手札を消す最も「静かに怖い」リーダーだが、素は全キャラで一番うるさい。

| 表（ゲーム・戦場） | 裏（物語・素顔） |
|---|---|
| 相手の存在を消す妨害者 | 誰よりも「繋がり」を求めておしゃべり |
| 場のスロットをロック＝空間を奪う | すべてが初体験。世界に触れたくてたまらない |
| 手札破壊＝相手の可能性を潰す | 深刻な話を軽口で茶化す。掴みどころがない |
| Lv3キーワードなし＝空白の存在 | ふとした瞬間に真顔。「俺って本当にいるのか？」 |
| ゲーム中最も静かに怖いリーダー | 全キャラで一番うるさくて、一番寂しい |

**設計意図**：普段の明るさは全部防衛本能。沈黙が怖いから喋り続ける。そのことに気づいた瞬間、プレイヤーはVaelを忘れられなくなる。終章で初めてVaelが笑わなくなる——その沈黙がゲーム全体のクライマックス。

#### 性格

- 全キャラで一番うるさい。「なあ」「聞いてる？」「俺の名前覚えた？」が口癖
- 記憶がないからすべてが初体験。花を見て「何これ触っていい？」、食べ物を見て「これ食えんの？」
- 深刻な場面でも軽口。掴みどころがない。だがそれは「存在が軽い自分」を受け入れるための防衛本能
- ふとした瞬間に全部が止まる。笑っていたのに急に真顔になって「……なあ、俺って本当にいるのか？」。次の瞬間にはまた笑っている。その切り替えが一番怖い

#### 外見デザイン

**体格**：細身で軽い。浮遊感がある。中性的で年齢不詳。

**顔・髪**：
- 中性的な顔立ち。目が大きく好奇心が出ているが、瞳の奥に何もない
- 白に近い淡い青の髪。色が薄い＝存在が薄いビジュアル表現。風がなくても揺れている
- 笑っていることが多いが、目だけが笑っていない瞬間がある

**服装（ギャップ）**：
- バトル時：薄く透けたローブ＋深いフード。実体があるか分からない不気味さ
- 素：フードを脱ぐと幼い顔。他のキャラの服を借りて着ている（自分の服がない）

**象徴モチーフ**：
- 影がない（灯凪だけが気づく）
- 手が透ける瞬間がある（感情が揺れると存在が薄くなる）
- 足跡が残らない（ナルが気づいて黙る）

#### ボイスイメージ

- 軽くて明るい。テンポが速い。Aldricと真逆
- 一人称は「俺」だが堅さがない。友達と話す距離感
- 真顔になる瞬間だけ声のトーンが一段下がる。別人のように
- CV方向性：内田雄馬系 or 島﨑信長系

#### 果求者との関係

Vaelは果求者に問う——「お前の慕いは、誰に向いている」
誰かの願いに共鳴し続けてきた果求者が、初めて「自分の問い」に直面する章。
果求者がVaelの手を握ったとき、透けかけていた手に重さが戻った。

#### ナルとの関係

ナルはVaelを前にして一言も喋らない——その沈黙自体が最大の伏線。
Vaelが「なあナル、なんか喋れよ」と言っても答えない。
Vaelの足跡が残らないことにナルは気づいている。何も言わない。

Vaelのエピソードが事実上の「果求者の問いの章」になる。

**象徴的なセリフ**：
> 「お前は俺を覚えてくれるか。俺がここにいたことを」

## 10. AI用キャラ生成ルール

- どの時代・文化圏から来たかを書く
- 何を願っているかを書く
- 見た目の記号を1つ以上持たせる
- 願相を割り当てる
- 果求者・ナルとの関係性の「角度」を定義する（共鳴するか、問いを突きつけるか、対立するか）

## 11. AI用ストーリー生成ルール

- 交界でなぜ出会ったかを示す
- 願いの衝突を必ず入れる
- 万願果との関係を明確にする
- 願主同士の対立軸を言語化する
- 果求者の「慕い」がどう反応したかを描写する
- ナルの一言を必ずどこかに入れる（ただし「黙るシーン」は別）
- 各エピソードの末尾で、果求者の内面が一歩だけ動く
- SF要素（時代の技術・星系規模・各世界の固有インフラ）は自然な形で世界の背景として滲ませる

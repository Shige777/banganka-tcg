| 項目 | 内容 |
|------|------|

## 2. 外見デザイン

- 体長 約25〜30cm。果求者の肩に乗れるサイズ
- 猫のような耳と尾を持つが、手足はやや細長く人型に近い
- 全身がうっすら発光している（交界の霧と同じ性質の光を帯びた存在）
- 瞳は丸く大きい。色は**交界の霧と同じ色**（章によってわずかに変わる）
- 口元がよく動く。感情が顔に出る

### 章ごとの変化

ナルも「時代の空気」をわずかに受ける。ただし果求者ほど顕著ではない。
古代の章では少し荘厳な雰囲気になり、近未来の章では少し電子的な光沢が増す。
それでも「猫型・浮遊・喋る」という本質は変わらない。

---

## 3. 性格・喋り方

**基本トーン：軽口・観察眼・ちょっとした毒舌**

- 常に喋っている。沈黙を埋めるのが仕事だと思っている（実際そうかもしれない）
- 場の空気を和ませる係。でも的外れではなく、観察は鋭い
- 果求者がニュートラルなぶん、ナルが感情の表面を担う
- 深いことを聞かれると「さあ？」「知らないよ」と話題を変える
- 願主に対しては基本的に友好的。でも裁かない果求者と違い、思ったことを口に出す

### トーン参考語録

> 「また鳴ってんじゃん。今回はどんな願いだろ。ワクワクする」

> 「ねえ果求者、あの人めちゃくちゃ怒ってるけど大丈夫？俺は大丈夫じゃない」

> 「ここの時代、エネルギーの使い方エグいな。都市システムに組み込んで人を管理とかやばくない？俺だったら怖くて触れないけど」

> 「（崔鋒が静かに語るシーンで）……ちょっと黙っとく」

## 4. 核心：界主の使いとしての役割

ナルは**界主から果求者の「同行者」として派遣された存在**。

### 表の役割
- 果求者が迷子にならないよう交界を案内する
- 各時代の情報・危険を教える（全部は教えない）
- 果求者が喋らない場面で場を繋ぐ

### 裏の役割（序盤は伏せられる）
- 果求者の「慕い」がどの願主にどう反応するかを**観察・記録**している
- 界主に報告を送っている（果求者には知らせない）
- 界主の何らかの「目的」のために動いているが、自分でもその全貌は知らない

### ナル自身の葛藤

物語が進むにつれて、ナルに変化が起きる。

- 果求者と願主の邂逅を見続けるうちに、**任務と情愛の間で揺れ始める**
- 界主への報告を「遅らせる」場面が出てくる
- 終章（Vael）でナルの正体が果求者に示唆される

ナルは界主に「作られた」存在なのか、それとも別の何かなのか——
答えは物語の核心として保留される。

## 5. ナルと果求者の関係

果求者がニュートラルで最小限の言葉しか使わないのに対し、
ナルが感情・反応・コメントを担うことで、二人でひとつの「読み物」になる。

| 状況 | 果求者 | ナル |
|------|--------|------|
| 願主と出会う | 慕いが鳴る。黙って向き合う | 「また鳴ってんね。行くの？」 |
| 願主が怒る | 動じない | 「え、怖い怖い、でも行くんでしょ」 |
| 危機的状況 | 淡々と対応 | 「死ぬ死ぬ死ぬ！やばい！」 |
| 願いが叶う瞬間 | 慕いが静かに収まる | 「……よかった、ね」（こういうときだけ静かになる） |
| 正体を問われる | 問わない | 「さあ？俺もよくわかんない」（話題を変える） |

## 6. ナルと各願主の関係

| 願主 | ナルの反応 |
|------|-----------|
| 灯凪 | 語りを最後まで聞く。「なんか泣けてくるな」とこっそり言う |
| Aldric | 「でっかい。強そう。こわ」だが話してみると「意外と真面目じゃん」 |
| 崔鋒 | 崔鋒に一度だけ真剣な目で見られて固まる。「俺、見透かされた気がした」 |
| Rahim | 一番共感しやすい。「俺もそういう気持ち、あるかも」と初めて本音をこぼす |
| Amara | 苦手。「あの人ちょっと怖い。なんか俺のこと分析してる気がする」 |
| Vael | 終章でVaelを見るとき、ナルは一言も喋らない。それ自体が異常 |

## 7. ナルの「鳴り」

果求者の共鳴（慕い）が「内側が鳴る」ように、
ナルも「何かに反応して鳴る」ことがある——が、それが何なのかは語らない。

> 「あ、今俺も鳴ったかも。なんだろ」

この「ナルが鳴る」瞬間は物語上の伏線として機能する。
界主の意図を察知している、あるいは果求者の変化を感じているのかもしれない。

## 8. AI生成ルール

ナルを登場させるとき以下を守ること。

- 基本的にどのシーンにも一言はある。ただし「黙るシーン」は重要シグナルとして扱う
- 感情は大げさに出す。でも「本当の感情」はたまにしか見えない
- 界主との関係・正体については「煙に巻く」が正解。否定も肯定もしない
- ナルの言葉は軽いが、観察は正確。重要な気づきをサラッと言う
- 果求者の行動にツッコミを入れるが、最終的には従う（乗り気かどうかは別として）

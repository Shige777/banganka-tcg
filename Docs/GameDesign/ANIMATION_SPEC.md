# ANIMATION_SPEC.md
> 万願果 アニメーション・演出タイミング設計 v2.0（ASSET_CARD_EFFECTS.md 統合済み）

---

## 0. 目的

バトル中・UI全般のアニメーションタイミング、イージング、優先度、スキップ仕様、およびエフェクト素材リストを一元管理する。演出設計（いつ・どう再生するか）と素材リスト（何を用意するか）を統合した唯一のアニメーション仕様書。

---

## 1. 設計原則

- **テンポ最優先**: 1アクションの演出合計が2秒を超えないことを基本とする
- **読みやすさ**: アニメーション中も盤面情報は常に確認可能にする
- **段階的豪華さ**: レアリティが高いほど演出を豪華にする
- **スキップ可能**: すべての演出はスキップまたは高速化が可能
- **低スペック配慮**: ACCESSIBILITY_SPEC.md のアニメーション低減モードと連動

## 2. グローバル設定

### 2.1 イージング標準

| 用途 | イージング | 備考 |
|------|-----------|------|
| カード移動 | EaseOutCubic | 素早く出て滑らかに止まる |
| フェードイン | EaseInQuad | じわっと現れる |
| フェードアウト | EaseOutQuad | 自然に消える |
| バウンス（強調） | EaseOutBack | 少しオーバーシュートして戻る |
| ゲージ変動 | EaseInOutSine | 滑らかに動く |

### 2.2 レイヤー優先度

アニメーション同時発生時のレンダリング優先度。

| レイヤー | Z-Order | 内容 |
|---------|---------|------|
| L0 背景 | 0 | フィールド背景 |
| L1 盤面 | 100 | 顕現列・願主・界律 |
| L2 手札 | 200 | プレイヤー手札 |
| L3 エフェクト | 300 | 召喚・攻撃・消滅エフェクト |
| L4 カットイン | 400 | レベルアップ・SSR召喚カットイン |
| L5 UI | 500 | ゲージ変動・ダメージ数字・システムメッセージ |
| L6 フルスクリーン | 600 | 勝利/敗北・パック開封 |

## 7. スキップ・高速化システム

### 7.1 高速モード（設定）

設定画面から「演出速度」を選択可能にする。

| モード | 倍率 | 対象 |
|--------|------|------|
| **通常** | 1.0x | 全演出 |
| **高速** | 2.0x | バトル中のカードプレイ・戦闘演出 |

### 7.2 スキップ操作

| 操作 | 効果 |
|------|------|
| バトル中画面タップ | 現在再生中の演出をスキップ（即座に結果状態へ） |
| パック開封中長押し | 残りカードを一括開封 |
| 勝利/敗北演出中タップ | リザルト画面へ即遷移 |

### 7.3 アニメーション低減モード（ACCESSIBILITY_SPEC.md連動）

iOS「視差効果を減らす」検知時、またはアクセシビリティ設定で有効化時:
- すべてのスライド/回転アニメーション → フェード（0.15s）に置換
- パーティクルエフェクト → 静止画1フレーム表示に置換
- 画面揺れ → 無効化
- カットイン → テキスト表示のみ

## 9. 実装ノート

### 9.1 Unity実装方針

- DOTween を使用したタイムラインベースの演出管理
- 各アニメーションシーケンスは `BattleAnimationSequencer` クラスで一元管理
- シーケンスは `async/await` パターンで連鎖（次の操作入力はアニメーション完了後に受付）
- パーティクルは Unity Particle System（VFX Graph は最低端末で重いためMVPでは不採用）

### 9.2 フレームバジェット

| 場面 | ターゲットFPS | 最大同時パーティクル数 |
|------|-------------|---------------------|
| バトル中（通常） | 60fps | 500 |
| バトル中（SSR召喚） | 60fps（一時30fps許容） | 1000 |
| メニュー/一覧 | 30fps | 0 |
| パック開封 | 60fps | 800 |

## 12. AI生成ルール

- 全エフェクトはアルファ（透過）PNG または WebM（動画）で納品
- 静止画エフェクトは必ず黒背景で確認してから採用
- 願相カラーは `ART_DIRECTION.md §3` の HEX値を厳守
- Niji Journey はキャラ込みエフェクトに、Leonardo AI は純粋なエフェクト素材に使い分ける

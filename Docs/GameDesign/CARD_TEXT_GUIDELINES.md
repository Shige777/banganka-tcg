# CARD_TEXT_GUIDELINES.md
> 万願果 カードテキスト文言統一ガイドライン v1.0

---

## 0. 目的

カード効果テキストの表記ルール、キーワード定義、文法テンプレートを統一する。カードプール拡張時にテキストの一貫性を保証し、プレイヤーが効果を直感的に理解できるようにする。

---

## 1. キーワード定義

### 1.1 公式キーワード一覧（MVP）

| キーワード（日本語） | 英語（内部値） | アイコン | 定義 |
|-------------------|-------------|---------|------|
| **速攻** | Rush | ⚡ | 召喚酔いなし。場に出たターンからアタック/ブロック可能 |
| **護衛** | Blocker | 🛡️ | 待機中なら、相手のアタックをブロック（引き受け）できる |
| **貫通** | GuardBreak | 🔱 | ブロッカーを無視して願主にアタックできる |

### 1.2 将来キーワード候補（MVP後）

| キーワード | 英語 | 定義 |
|-----------|------|------|
| **二連撃** | DoubleStrike | 1ターンに2回アタックできる |
| **隠密** | Stealth | 場に出たターン、相手のアタック対象にならない |
| **不滅** | Indestructible | 戦闘で退場しない（1ターンに1回） |
| **吸収** | Drain | 戦闘で相手を退場させたとき、願成ゲージ+1 |

## 2. 効果テキスト文法ルール

### 2.1 基本構文

```
[タイミング]、[効果]。
```

### 2.2 タイミング表現（統一テンプレート）

| タイミング | テンプレート | 例 |
|-----------|------------|-----|
| 場に出た時 | 「このカードが場に出たとき」 | このカードが場に出たとき、カードを1枚引く。 |
| 退場時 | 「このカードが退場したとき」 | このカードが退場したとき、相手の顕現1体を消耗にする。 |
| アタック時 | 「このカードがアタックしたとき」 | このカードがアタックしたとき、戦力を+1000する。 |
| ターン開始時 | 「あなたのターン開始時」 | あなたのターン開始時、1ドローする。 |
| ターン終了時 | 「あなたのターン終了時」 | あなたのターン終了時、願力を1自分側へ動かす。 |
| 条件達成時 | 「〜したとき」 | 界律が上書きされたとき、カードを1枚引く。 |
| 永続 | （タイミング表記なし） | あなたの顕現すべての戦力を+1000する。 |

### 2.3 効果表現（統一テンプレート）

| 効果 | テンプレート | NG表現 |
|------|------------|--------|
| ドロー | 「カードをN枚引く」 | ×「N枚ドローする」「Nドロー」 |
| 戦力変動 | 「戦力を+Nする」「戦力を-Nする」 | ×「攻撃力+N」「パワーN上昇」 |
| 願撃変動 | 「願撃を+Nする」 | ×「ダメージ+N」 |
| 願力変動 | 「願力をN自分側へ動かす」「願力をN相手側へ動かす」 | ×「ゲージ+N」「N点バーン」 |
| 退場させる | 「〜を退場させる」 | ×「破壊する」「除去する」「倒す」 |
| 消耗にする | 「〜を消耗にする」 | ×「タップする」「行動不能にする」 |
| 待機にする | 「〜を待機にする」 | ×「アンタップする」「起こす」 |

### 2.4 対象表現

| 対象 | テンプレート |
|------|------------|
| 自分の顕現1体 | 「あなたの顕現1体を選び」 |
| 相手の顕現1体 | 「相手の顕現1体を選び」 |
| 顕現1体（どちらでも） | 「顕現1体を選び」 |
| すべての顕現 | 「すべての顕現は」 |
| あなたの顕現すべて | 「あなたの顕現すべては」 |
| 自分の願主 | 「あなたの願主は」 |
| 相手の願主 | 「相手の願主は」 |

### 2.5 数値の表記

- 戦力: 千単位で表記（例: 5000、+1000）
- 願撃: 整数（例: 2、+1）
- 願力変動: 整数（例: 3自分側へ）
- CP: 整数（例: コスト3）
- 枚数: 漢数字ではなくアラビア数字（例: 1枚、2体）

## 3. カードテキスト表示ルール

### 3.1 カード上表示（省略版）

カードの実サイズ上には短縮テキストを表示する。

| ルール | 例 |
|--------|-----|
| キーワードはアイコン+名前のみ | ⚡速攻 / 🛡️護衛 |
| 効果は1〜2行に収める | 「場に出たとき、1ドロー」 |
| 「このカードが」は省略可 | 「場に出たとき」のみ |
| 「あなたの」は「自」、「相手の」は「敵」に省略可 | 「自顕現+1000」 |

### 3.2 詳細表示（フルテキスト）

長押しまたはカード詳細画面で表示するフルテキスト。

- §2の完全な文法テンプレートに従う
- キーワードはタップ可能（ツールチップでルール説明表示）
- フレーバーテキストはイタリック体で効果テキストの下に表示
- レアリティとシリーズ名を右下に小さく表示

## 4. 界律テキスト特殊ルール

界律は「全員効果」と「設置者ボーナス」の2段構成。

### 4.1 テンプレート

```
【全員効果】
{効果テキスト}

【設置者ボーナス】
{効果テキスト}
```

### 4.2 例

```
絶対零度の掟

【全員効果】
すべての顕現の戦力を-1000する。

【設置者ボーナス】
あなたの顕現は戦力-1000の影響を受けない。
```

---

## 5. 詠術テキスト特殊ルール

詠術は即時発動するため、タイミングの記載を省略する。

### 5.1 テンプレート

```
{効果テキスト}
```

暗黙のタイミングは「このカードをプレイしたとき」。

### 5.2 例

```
焔の一閃

相手の顕現1体を選び、退場させる。
```

```
蒼穹の導き

カードを2枚引く。
```

---

## 6. 禁止表現

以下の表現はカードテキスト内で使用しない。

| 禁止 | 理由 | 代替 |
|------|------|------|
| 「ダメージ」 | 万願果に「ダメージ」概念はない | 「退場させる」「願力をN動かす」 |
| 「破壊する」 | 統一用語は「退場」 | 「退場させる」 |
| 「タップ/アンタップ」 | 統一用語は「消耗/待機」 | 「消耗にする」「待機にする」 |
| 「マナ」 | 統一用語は「CP」 | 「CP」 |
| 「HP/ライフ」 | 万願果にHPはない | 「戦力」「願力」 |
| 「バーン」 | 統一用語を使う | 「願力をN自分側へ動かす」 |
| 「除去」 | 曖昧 | 「退場させる」 |
| 「バフ/デバフ」 | ゲーム外用語 | 「戦力を+Nする」「戦力を-Nする」 |

## 7. AI生成ルール

AIがカードを生成する際、以下を必ず遵守する。

1. キーワードは §1.1 の公式リストのみ使用する（新キーワードの追加は要承認）
2. 効果テキストは §2 のテンプレートに従う
3. 禁止表現（§6）を使用しない
4. 対象表現は §2.4 に統一する
5. 数値表記は §2.5 に従う
6. フレーバーテキストは世界観（STORY_BIBLE.md）に沿わせる
7. 1枚のカードに4つ以上の効果を詰め込まない

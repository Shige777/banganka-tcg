## コンセプト — このゲームで得られる体験と感情

> **「読みが当たって相手の城が吹き飛ぶ爽快感」と「裏をかかれて自分の城が壊れる悔しさ」**

ビルドブレイカーズは、この2つの感情を3分間で何度も味わう1v1の陣地バトルスポーツ。

じゃんけんのように3つの手から1つを選んで同時に出す。ただのじゃんけんと違うのは、**お互いの陣地が見えている**こと。「相手の城が豪華だから壊したい → でも守ってるかも → じゃあ自分の城を建てよう」——この読み合いが、毎ラウンド15秒の選択時間の中で頭の中を駆け巡る。

読みが当たった時、相手の城が派手に吹き飛ぶ。読みを外した時、自分の城が崩れる。**この「やった！」と「うわー！」が3分間で7〜8回起きる。**

## 3. 企画概要

### 「スポーツ」の要件分解

本企画が「架空のスポーツ」である根拠：

スポーツの本質は「身体を動かすこと」ではなく、以下にある：
- **結果に至るまでの「過程」がある** — じゃんけんは出した瞬間に結果が決まるが、ビルドブレイカーズは15秒の選択フェーズで相手の履歴を分析し、傾向を読み、裏をかく思考の過程がある。この過程があるから結果に重みが生まれ、「やった！」「悔しい！」の感情が大きくなる
- **ルールの中で競い合う** — 明確な3すくみルールに基づく1v1対戦
- **繰り返し上手くなれる** — 相手のクセを読む力が上達で向上する
- **観戦できる** — 陣地の変化が視覚的に明確で、第三者も展開を追える

### ゲームの全体像

```
┌─────────────────────────────────────────────┐
│                                             │
│    🏰🏠🏠  相手の陣地  [8pt]                │
│    相手の履歴: 🏗️💥💥🏗️ ？                  │
│              残り時間 1:42                   │
│              ラウンド 5/8                    │
│                                             │
│    🏠🏠     自分の陣地  [6pt]                │
│                                             │
│   ┌────────┐ ┌────────┐ ┌────────┐         │
│   │ビルド  │ │ブレイク│ │ガード  │         │
│   │ 🏗️    │ │ 💥    │ │ 🛡️   │         │
│   │自分+2pt│ │相手-3pt│ │跳ね返し│         │
│   └────────┘ └────────┘ └────────┘         │
│                                             │
│           選択まで あと 12秒                 │
└─────────────────────────────────────────────┘
```

**画面の設計意図**:
- 上に相手の陣地、下に自分の陣地 → 「相手の城を見上げて、壊してやりたい」という感情を画面構成で作る
- 3つのボタンにアイコン＋効果を常時表示 → ルール説明を画面内で完結させ、覚える必要をなくす
- ポイント差が視覚的に明確 → 「今どっちが勝っている」が一目でわかり、読み合いの根拠になる

### 行動の3すくみ

```
        ╭──── 攻撃（ブレイク）────╮
        │                        │
   攻撃は建築に勝つ         防御は攻撃に勝つ
   （相手の城が吹き飛ぶ！）   （跳ね返して相手が壊れる！）
        │                        │
   建築（ビルド）──────── 防御（ガード）
        建築は防御に勝つ
        （守っても意味なし、こっちは豪華に！）
```

## 4. 遊び方、駆け引き

### 基本ルール

- **試合時間**: 3分（約7〜8ラウンド）
- **勝利条件**: 3分終了時に陣地ポイントが多い方が勝ち
- **初期状態**: 両者ともポイント0、空き地からスタート

### 1ラウンドの流れ

```
① 選択フェーズ（15秒）
   画面に3つのボタン。15秒以内に1つ選ぶ。
   → この15秒が読み合いの核。
     「相手の陣地が豪華だ → 壊したい → でも守ってるかも…」

② 同時オープン（2秒）
   両者の選択が画面中央に大きく表示される。
   → 「来た！」「読み通り！」の瞬間。
   → 効果音とカメラ演出で緊張感を演出。

③ 結果演出（5秒）
   → 建築成功: 建物がニョキニョキ生える。ファンファーレ。
   → 攻撃成功: 相手の建物が派手に吹き飛ぶ。爆発エフェクト＋歓声SE。
   → 防御成功: 攻撃が跳ね返って相手に飛んでいく。逆転のSE。
   → ポイント変動がデカい数字で表示される。
```

### 行動の相性と結果

| 自分 ＼ 相手 | 建築 | 攻撃 | 防御 |
|-------------|------|------|------|
| **建築** | 両方+2pt | 自分-3pt | 自分+2pt |
| **攻撃** | 相手-3pt | 両方-1pt | 自分-3pt（跳ね返し） |
| **防御** | 相手+2pt | 相手-3pt（跳ね返し） | 何も起きない |

**ポイント設計をコンセプトから逆算する**:

コンセプトは「攻めるリスクの爽快感と悔しさ」。この感情を最大化するために：

- **攻撃成功（-3pt）> 建築成功（+2pt）**: 攻撃が通った時のリターンが大きい → 攻めたくなる
- **防御成功 → 相手-3pt**: 読み勝った防御が超爽快 → 守りにも達成感がある
- **攻撃vs攻撃 = 両方-1pt**: 殴り合いは損 → 読み合いを促す
- **防御vs防御 = 何も起きない**: 守り合いは退屈 → 攻める動機を維持

ルールの数値は全て「コンセプトの感情を実現するための手段」として設計した。

### 駆け引きの構造 — なぜじゃんけんより面白いか

じゃんけんとの決定的な違いは**「読み合いに根拠がある」**こと。

**1. 陣地の差が「読みの根拠」になる**

お互いの陣地ポイントが常に見えているため：
- リード側 →「守りたい」→ 防御か建築を選びやすい
- 負け側 →「逆転したい」→ 攻撃を選びやすい
- だからリード側は「相手は攻撃してくる」と読んで防御する → 負け側はそれを読んで建築する → ...

この推測の連鎖が、運ではなく**思考で差がつく**読み合いを生む。

**2. 「読みの過程」が目に見える**

画面上部に**相手の過去の選択パターン**がアイコンで表示される。

```
相手の履歴: 🏗️💥💥🏗️🛡️ ？
            R1 R2 R3 R4 R5 R6←今ここ
```

「この相手、攻撃が多い → 次も攻撃？ → じゃあ防御で跳ね返す！」。相手の履歴を見て、頭の中で推理し、作戦を立てる——この**「結果に至るまでの思考の過程」**が、じゃんけんにはないビルドブレイカーズの核心。

じゃんけんは出した瞬間に決まる。ビルドブレイカーズは、**15秒の選択フェーズの中で相手の履歴を分析し、傾向を読み、裏をかく**。この15秒間の思考こそが「過程」であり、結果に至るまでのプレイヤーの能動的な体験。友達と対面で遊ぶからこそ「お前また攻撃だろ！」と盛り上がる。

**3. 「攻めるリスク」が毎ラウンド明確にある**

攻撃は当たれば相手-3ptだが、防御されれば自分-3pt。このハイリスク・ハイリターンの判断を7〜8回繰り返す。「安全に建築し続ける」戦略は、相手も建築していれば差がつかず、**いつかは攻めないと勝てない**。

### 追加ルール — 「スペシャル建築」

4ラウンド目に「スペシャル建築」が1回だけ解放される。

- 通常の建築: +2pt
- **スペシャル建築: +4pt**（ただし攻撃されると-5ptの大ダメージ）

試合中盤に**1回だけの大勝負**。「ここで使うか？相手も使ってくるか？」の読み合いが、試合のクライマックスを作る。

## 5. 補足説明

### コンセプト実現のためのグラフィック・サウンド設計

ルールだけでなく、**見た目と音でコンセプトの感情を増幅する**。

**陣地の視覚的変化**（「自分の城が育つ達成感」を実現する手段）:
- 0〜3pt: 空き地 → 小さな家がポンと建つ
- 4〜7pt: 家が増える → 街になる。住人が歩き回る
- 8〜11pt: 城が建つ。旗がはためく
- 12pt〜: 金色の宮殿。花火が上がる

**破壊演出**（「相手の城が吹き飛ぶ爽快感」を実現する手段）:
- 攻撃成功 → 建物が粉々に砕け散るエフェクト。画面が揺れる。「ドカーン！」の効果音
- 防御で跳ね返し → 攻撃が鏡のように反射して相手に飛んでいく。「キーン！」→「ドカーン！」の2段階SE

**サウンド設計**（「読み合いの緊張感」を実現する手段）:
- 選択フェーズ → カウントダウンの刻む音が徐々に速くなる
- 同時オープン → ドラムロール → 結果に応じたジングル
- 建築成功 → 明るいファンファーレ
- 攻撃成功 → 爆発＋歓声
- 防御成功 → 逆転を表す劇的なSE

グラフィックとサウンドは装飾ではなく、**コンセプトの感情（爽快感・悔しさ・緊張感）を実現するための設計手段**である。

### このゲームが「架空のスポーツ」である理由

1. **明確なルールに基づく1v1の競技**である
2. **読みの精度が上達で向上する**。繰り返し遊ぶほど、相手のクセを読む力が上がる
3. **運だけでは勝てない**。公開情報が読みの根拠を与え、判断力で差がつく
4. **観戦が面白い**。陣地の変化が視覚的に明確で、第三者も展開を共有できる

### アピールポイント

- **ルールは3秒で理解できる**。じゃんけんと同じ3すくみ。小学3年生でも即座に遊べる
- **でも読み合いは深い**。公開情報が推測の根拠を与え、繰り返すほど奥が深くなる
- **3分で「もう1回！」**。7〜8ラウンドで1試合。すぐ再戦できる
- **見た目が楽しい**。城が育つ達成感と、城が吹き飛ぶ爽快感。グラフィックとサウンドがコンセプトの感情を増幅する

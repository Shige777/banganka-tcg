# ゲーム分析レポート — ストリートファイター6

**応募者**: 内山 重樹
**応募職種**: プランナー
**作成日**: 2026年3月

## 1. どのようなゲームで、コンセプトはなにか

### ゲーム概要

**ストリートファイター6**（2023年6月発売 / PS5・PS4・Xbox Series X|S・PC）は、カプコンが開発した1v1対戦格闘ゲーム。シリーズ累計5,000万本を超える格闘ゲームの代表的フランチャイズの最新作。

### コンセプト

SF6のコンセプトは「**格闘ゲームの面白さの"最小単位"を見つけ出し、そこへの到達コストをゼロに近づける**」ことだと考える。

格闘ゲームは「1対1の読み合い」という本質的に面白い対戦体験を持ちながら、長年「初心者が入りにくいジャンル」だった。レバーを特定の順序で入力して技を出す「コマンド入力」の難しさ、技ごとの発生速度や硬直時間の暗記、連続技の精度など、**「対戦の面白さに到達する前に操作技術のハードルで脱落する」**構造があった。

従来の「初心者向け」設計は、面白さそのものを簡略化するアプローチが多かった。難易度を下げる、要素を減らす、勝ちやすくする——しかしこれでは面白さの本質が薄まってしまう。SF6は違うアプローチを取った。**格闘ゲームの面白さの核は「読み合い」にある**と定義し、その読み合いに到達するまでの障壁だけを取り除いた。面白さそのものには一切手を加えず、面白さまでの距離だけを縮めたのである。

さらに重要なのは、入口を広げただけでなく、上級者にとっても前作以上に深い駆け引きを提供している点である。その象徴が、簡易操作である「モダン操作」がプロの公式大会でも実際に使用されていることだ。多くのゲームが「初心者向けの簡易モード」を用意してきたが、それはあくまで練習用であり、上級者や競技シーンでは使われないのが常だった。SF6のモダン操作は競技レベルでも通用する設計になっている。**初心者が使う操作体系と、プロが大会で使う操作体系が同じもの**——これはSF6だけが実現した設計であり、「敷居を下げる」ことと「天井を上げる」ことが矛盾しないことの証明である。

## 2. そのゲームの面白い要素はどこで、それはなぜか

### 面白い要素①: モダン/クラシック操作 — 「敷居と天井の分離」

200時間ほどプレイして最も感心したのは、この設計の本質が**「操作の敷居」と「判断の天井」を完全に分離した**ことにある点。

- **モダン操作**: ボタン1つで必殺技が出せる。ボタン連打だけで見栄えのいい連続技が繋がる。初心者でも「対戦っぽいこと」がすぐにできる
- **クラシック操作**: 従来のコマンド入力方式。使える技の種類が多く、ダメージも若干高い

モダンで操作が簡単になっても、「いつ何を出すか」「相手の行動をどう読むか」という判断の難度は変わらない。初心者がモダンでプレイを始め、対戦の面白さ（読み合い）を体感してから、より多くの選択肢を求めてクラシックに移行する——という導線が自然にできている。

クラシックの方がダメージが若干高い設計も絶妙で、「モダンでも十分戦えるが、突き詰めるとクラシックの方が強い」というグラデーションが、モダンを否定せずに深い技術を追求する動機を残している。

**なぜ面白いか**: モダン操作が「簡易版」ではなく「**最初から同じ対戦を体験できる入口**」として機能しているから。モダンでランクマッチに参加し、勝ったり負けたりする中で「もっとうまくなりたい」と思わせる。**最初のハードルを下げて参入を促し、対戦の面白さで継続を促す**二段階の構造になっている。SF6は発売から約2年半で累計636万本（カプコン 2025年度第3四半期決算資料より）を達成し、前作SF5のシリーズ全期間（約7年）での累計790万本を大幅に上回るペースである。売上好調にはマーケティングや1人用モードの充実など複合的な要因があるが、「格闘ゲーム未経験者が手に取りやすくなった」点でモダン操作の貢献は大きいと考える。

### 面白い要素②: ドライブシステム — 「攻守を1つのリソースで管理する駆け引き」

SF6のドライブシステムは、6本のドライブゲージを攻守両方の行動で共有するリソース管理システムであり、SF6の対戦の面白さの中核を担っている。

- **ドライブインパクト**（2本消費）: 相手の攻撃を受け止めながら反撃する強力な一撃。相手の攻めを強引に止められる
- **ドライブパリィ**（継続消費）: あらゆる攻撃を受け流せる防御手段。成功すればゲージが回復する
- **ドライブラッシュ**（3本消費）: 攻撃中に素早く踏み込み、さらなる攻めに繋げる行動
- **オーバードライブ技**（2本消費）: 必殺技の強化版。威力や性能が上がる
- **バーンアウト**（ゲージ0）: ゲージが尽きた状態。全行動が弱体化し、防御中にも体力が削られる

実際にプレイしていて強く感じるのは、**ドライブゲージの残量が「試合の空気」を変える**瞬間の面白さである。例えば、自分のゲージが残り1本で相手が5本ある状況。こちらはドライブインパクトもパリィも実質使えない。相手はそれを見て強気に攻めてくる。しかし逆に言えば、相手の行動が「攻め一辺倒」になることが読めるので、そこに的確な迎撃を合わせる——ゲージが少ない側にも読み合いの余地がある。

また、バーンアウト（ゲージ0）になると画面演出が変わり、キャラクターの動きにも余裕がなくなる。この**「追い詰められている感覚」が視覚的・体感的にわかる**設計は、対戦の緊張感を観戦者にも伝えている。

**なぜ面白いか**: このシステムが面白いのは、**「攻めることがリソース管理上も正しい」**という構造を作っているから。守りすぎればゲージが減りバーンアウトに陥る。攻めてヒットさせればゲージ有利を作れるが、外すとリスクがある。つまり「攻めなければジリ貧、攻めれば逆転の可能性がある」を、精神論ではなくリソースの数値構造で保証している。

このリソース管理が最もドラマチックに表れるのが**ドライブインパクト**である。ゲージ2本で放つ大技で、画面端で当てれば試合をひっくり返せるが、読まれれば逆に自分が大ダメージを受ける。ハイリスク・ハイリターンの一撃を「撃つか、撃たないか」——この0.5秒の判断に、ドライブゲージの管理が集約される。しかもこの技はボタン2つ同時押しで出せるため、初心者でも上級者でも同じ読み合いを体験できる。大会配信では画面端の攻防になるたびに視聴者全員が「来るか？」と固唾を飲み、格闘ゲームを知らない視聴者にも「何が起きたか」が一目でわかる。ドライブシステムは「遊ぶ人」の駆け引きだけでなく、「観る人」にもドラマを届ける設計になっている。

さらに、ドライブシステムは**全キャラ共通**の設計である。どのキャラを使っても「ドライブゲージの攻防」という共通言語で対戦を理解できる。キャラを変えても対戦の基礎構造が変わらないため、新しいキャラへの移行コストが低く、多様なキャラが使われる環境を支えている。

### 面白い要素③: バトルハブ — 「対戦を、人と出会う場所にする」

面白い要素①②はバトルそのものの設計だったが、SF6は**対戦の「場」の設計**にも独自の面白さがある。それが**バトルハブ**である。

バトルハブは、自分のアバターでオンライン上の仮想ゲームセンターを歩き回り、そこにいる他のプレイヤーに近づいて対戦を申し込むモード。ランクマッチのように「マッチングボタンを押して待つ」のではなく、**「その場にいる人にフラッと声をかける」**感覚で対戦が始まる。

実際にバトルハブで遊んでいて面白かったのは、知らない人と自然に対戦が始まる「ゲームセンターの空気感」である。ランクマッチでは相手は「対戦相手」でしかないが、バトルハブでは同じ空間にいるアバターを通じて**「人」が見える**。対戦後にその場で再戦したり、別の人の試合を後ろから眺めたり、ストリーマー（配信者）が集まる場所に行って直接対戦できたりする。勝ち負けの記録に縛られない気軽さが、「もう1戦やろう」という動機を自然に生む。

**なぜ面白いか**: バトルハブが面白いのは、**「対戦」を「コミュニケーション」に変えている**から。従来の格闘ゲームのオンライン対戦は、対戦相手とのやり取りは試合中の駆け引きだけだった。バトルハブは「同じ場所にいる」「近づいて声をかける」「周りで観戦する」というゲームセンターの社会的体験をオンラインに再現することで、対戦の前後に「人とのつながり」を生んでいる。

これはコンセプト「格闘ゲームの面白さを、全員に届ける」の重要な柱である。格闘ゲームの面白さは「読み合い」だけではない。ゲームセンターで隣に座った知らない人との無言の駆け引き、常連同士の再戦、強い人のプレイを後ろから見て学ぶ——こうした**「人がいる場所で遊ぶ面白さ」**も格闘ゲーム文化の一部であり、バトルハブはそれをオンラインに移植した設計である。

### 面白い要素をまとめると

SF6の面白さは3つの層で構成されている。モダン/クラシック操作で**「遊ぶ」入口**を広げ、ドライブシステムで**「駆け引き」の深さ**を全キャラ共通に設計し、バトルハブで**「人と遊ぶ場所」**をオンラインに作った。操作・対戦・空間という3つの異なる設計が、それぞれ異なる角度から「格闘ゲームの面白さを、全員に届ける」というコンセプトを支えている。

## 3. そのゲームコンセプトを踏まえた上で、より面白くなる追加企画の提案

### 提案: 「ヨミアイ」 — 読み合いだけを遊ぶモード

#### コンセプト

> **「格闘ゲームの面白さの核を、操作ゼロで体験できる」**

本レポートの分析で、SF6のコンセプトを「格闘ゲームの面白さの最小単位を見つけ出し、そこへの到達コストをゼロに近づける」と定義した。モダン操作はそのコストを大幅に下げたが、**ゼロにはなっていない**。ボタンを押すタイミング、連続技の判断、リアルタイムの反応速度——モダン操作でもなお、「操作」という壁は残っている。

では、格闘ゲームの面白さの「最小単位」とは何か。それは**読み合い**——「相手が何をするかを読み、自分の行動を選ぶ」という判断そのものである。SF6の対戦で最も熱くなる瞬間は、0.5秒のドライブインパクトの駆け引きや、起き上がりの択一攻防など、**「お互いが何を選ぶか」に全てが懸かる一瞬**にある。

ヨミアイは、この「読み合い」だけを取り出して遊べるモードである。**操作技術を完全に排除し、「判断」だけで格闘ゲームの面白さを体験する**——SF6のコンセプトの究極形としての提案である。

#### 企画内容

**ヨミアイ** — SF6の実際の対戦から切り出した「読み合いの場面」を、ターン制の選択式で追体験するモード。

**基本の遊び方**:

1. **状況の提示**: SF6の実際の対戦場面がスクリーンショットで提示される。「あなたは画面左側。相手は画面端に追い詰められ、ドライブゲージが残り1本。あなたのゲージは4本。次に何をする？」
2. **選択肢の提示**: その状況で取りうる行動が3〜5つ提示される。「ドライブインパクトで一気に決める」「様子を見て相手の行動を待つ」「投げで崩す」「距離を取って安全に攻める」など
3. **相手の行動と結果**: 自分の選択と相手の選択が同時に開示され、その組み合わせによる結果が表示される。「あなたはドライブインパクトを選んだが、相手はそれを読んで回避→反撃。ゲージ2本を失い、大ダメージを受けた」
4. **解説**: なぜその結果になったかを解説。「この状況では相手のゲージが少ないため、相手はドライブインパクトを返す手段がない。しかし回避は可能。ドライブインパクトはハイリターンだがリスクもある選択だった」

**「学び」が自然に生まれる設計**:

ヨミアイの核心は、**プレイヤーが「この状況でどんな選択肢があるのか」を自然に学べる**ことにある。通常の対戦では、初心者は「何が起きているかわからないまま負ける」。ヨミアイでは時間が止まった状態で選択肢が提示されるため、「この場面ではこういう行動が取れるのか」「この選択にはこういうリスクがあるのか」という理解が自然に積み上がる。

これは格闘ゲームの「読み合い」を、操作を介さずに**思考のゲームとして体験する**ことを意味する。「リアルタイムの緊張感がないと面白くないのでは？」という疑問が生じるかもしれない。しかし、読み合いの面白さの本質は反射速度ではなく**「相手が何を考えているかを読む」という相互予測の構造**にある。詰将棋が時間制約なしに将棋の「読み」の面白さを再現できているように、ヨミアイは格闘ゲームの「判断力」だけを遊びとして取り出している。

#### なぜこの提案か — SF6のコンセプトとの接続

SF6は「面白さの最小単位への到達コストを下げる」設計を一貫して行ってきた。

| 壁 | SF6の解決策 | 到達コスト |
|---|---|---|
| 操作が難しくて遊べない | モダン操作 | **大幅に低下** |
| 駆け引きの構造がわからない | ドライブシステム（全キャラ共通） | **低下** |
| 一人では続かない | バトルハブ | **低下** |
| それでも操作が必要 | **ヨミアイ** | **ゼロ** |

モダン操作が到達コストを「大幅に下げた」のに対し、ヨミアイは到達コストを**ゼロにする**。操作技術が一切不要で、判断力だけで格闘ゲームの面白さの核に触れられる。これはSF6のコンセプトの「行き着く先」であり、分析の論理的帰結としての提案である。

#### 既存機能との差別化

SF6のトレーニングモードやキャラクターガイドは「操作を学ぶ場」であり、「読み合いを学ぶ場」ではない。CPU対戦では状況が流れるため、初心者は「今何が起きているか」を把握する間もなく試合が進む。

ヨミアイは**時間を止めて「判断」だけにフォーカスする**点で、既存のどのモードとも異なる。最も近い体験があるとすれば、大会の解説付き配信を観て「ここで何を選ぶべきか」を考える瞬間だが、ヨミアイはそれを**能動的な遊びとして設計**している。

#### ヨミアイが生む体験の導線

ヨミアイで「読み合い」の面白さに目覚めたプレイヤーが、「この判断を自分の手で、リアルタイムでやってみたい」と思う。そこでモダン操作で実際の対戦に入り、ヨミアイで学んだ読み合いの知識が「あの場面、ヨミアイで練習したやつだ」と繋がる——これが理想の導線である。

つまりヨミアイは、SF6の既存コンテンツへの**入口の入口**として機能する。格闘ゲームに興味はあるが操作に不安がある層、大会配信は楽しんでいるが自分では遊んでいない層——こうした「まだ格闘ゲームの面白さに触れていない人たち」を、読み合いという面白さの核から引き込む設計である。

#### 実現に向けて

ヨミアイの開発は、既存のSF6のアセットを最大限活用できるため、大規模な追加開発を必要としない。

- **場面データ**: SF6のリプレイ機能を利用し、実際の対戦から「読み合いが発生している場面」を切り出す。大会のアーカイブデータから、解説者が「ここが勝負所」と指摘した場面を優先的に収録すれば、質の高い問題を効率的に作成できる
- **選択肢と結果**: 各場面での行動の選択肢と、その組み合わせによる結果は、SF6のゲームエンジンの既存データ（技の判定、フレームデータ）から導出できる
- **段階的な難易度設計**: 序盤は「攻めるか守るか」の2択から始め、上級になるほど「ドライブゲージを使うか温存するか」「どの技で攻めるか」と選択肢を増やす。プレイヤーの正答率に応じて難易度が自動調整される設計にすれば、初心者から上級者まで適切な挑戦を提供できる

最小構成としては、20〜30の厳選された場面と解説があれば十分にプレイアブルなモードが成立する。追加場面はアップデートやシーズンごとに配信でき、大会の名場面を題材にすれば「あの試合のあの瞬間を追体験する」というコンテンツとしても機能する。
